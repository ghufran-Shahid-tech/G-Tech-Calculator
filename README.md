## ğŸ“ Detailed Project Description

G-Tech Calculator was developed as a practical implementation of JavaScript fundamentals combined with structured UI design. The goal of the project is to create a fast, user-friendly, and visually appealing calculator that performs real-time arithmetic operations.

The application allows users to input numbers and operators either by clicking on on-screen buttons or by using their keyboard. The system dynamically updates the display area while processing mathematical expressions accurately.

The interface is intentionally minimal to ensure usability. Button spacing, alignment, and responsiveness are optimized for different screen sizes. Error handling prevents invalid operations such as dividing by zero or entering incorrect expressions.

This project strengthens understanding of:

- JavaScript expression evaluation  
- Event listeners and DOM interaction  
- Conditional logic  
- Responsive web design  
- UI/UX layout structure  

The calculator is lightweight, fast, and optimized for performance.
